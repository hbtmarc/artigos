# artigos
Projeto Web de Artigos

## Deploy (GitHub Pages)
- Settings -> Pages -> Source = GitHub Actions
- Acompanhe logs em Actions -> Deploy to GitHub Pages
- URL esperada: https://hbtmarc.github.io/artigos/

## Testes locais (Base path)
Comandos:
```
npm run dev
npm run build
npm run preview
```

URLs para validar:
- DEV: http://localhost:4321/artigos/ e http://localhost:4321/artigos/admin/
- PREVIEW: use a URL exibida pelo `npm run preview` e adicione `/artigos/admin/`
